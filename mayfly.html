The scientific method

Pre-registered. Falsifiable.

The hypothesis and pass/fail gates were fixed before the first trade. 30 trading days. Three gates. Public failures.

GATE 01Mechanical

Zero critical failures — no naked stops, no duplicate orders.

GATE 02Risk fidelity

Not one trade breaches the 1% / stop / −6% rules. Verified by SQL, not trust.

GATE 03Economics

Expectancy ≥ +0.05R · profit factor ≥ 1.1 · drawdown ≤ 8%.

Anti-cheating commitments
  • Frozen rules: any change to sizing, stops, or the breaker resets the 30-day clock to zero.
  • Frozen learning: the self-tuning feedback loop is suspended for the window, so it cannot overfit mid-test.
  • Failures are data: software incidents are logged, classified, and published. A hidden failure is a failed experiment.
  • Falsifiable: if any gate fails, the result is NO-GO — as useful as a pass.
Honest limitations
  • Market data is ~15 minutes delayed; bid/ask spreads are synthesized. Real-time execution would differ.
  • Simulated orders can't move the market; real orders in micro-caps do.
  • One 30-day window is a small sample. A pass is evidence, not proof.
